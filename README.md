# Projeto-Codifique

  O projeto Codifique, realizado por alunos do Instituto de Ciências Matemáticas e de Computação (ICMC) da USP, em São Carlos é um curso gratuito de programação básica oferecido para quem está no ensino médio ou no cursinho e deseja conhecer um pouco mais sobre computação. 
  Tive a sorte de ser um dos 40 participantes deste projeto e no final consegui programar o jogo "Pong" em Javascript, utilizando a ferramenta p5.js

https://p5js.org/
